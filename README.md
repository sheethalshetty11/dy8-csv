# Employee CSV Processor

## Description
Reads employee data from a CSV file and generates a summary report.

## Files
- csv_processor.py
- employees.csv
- app.log

## Run Instructions

1. Open terminal
2. Navigate to the project folder

Run:

python csv_processor.py

## Features
- Reads employee data using csv module
- Calculates employee statistics
- Department-wise employee count
- Exception handling
- Logging using app.log
